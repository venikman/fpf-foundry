# F.18 Name Card: Strategist Role

## 1. Twin-Labels
- **Technical ID**: `strategist-role`
- **Plain-English**: "Strategist Role"

## 2. Minimal Definitional Statement (MDS)
> The role that frames objectives, constraints, and acceptance criteria for an agent session and issues handoffs to executors.

## 3. Context of Meaning
Defined within the **AgentWorkflow** bounded context.

## 4. Rationale
Separates planning and acceptance from execution to keep the two-agent loop auditable and deterministic.
