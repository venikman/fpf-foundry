# Changelog

## 0.1.0

- Initial Bun-first `fpf` CLI published to npm (`bunx --bun @venikman/fpf ...`).
- Commands: `init`, `check`, `doctor`, `mint-name`, `record-drr`, `log-work` (all root-agnostic; `--json` supported).
- GitHub Actions CI (Linux/macOS/Windows) + automated releases with compiled binaries.
